# Studio Ornate
Este projeto foi desenvolvido para o Programa do Rio de Janeiro, Programadores Cariocas

## Uso

```bash
git clone https://github.com/moniquemoura/studio_ornate
cd studio_ornate
npm install
npm start
```

Em outro terminal
```bash
git clone https://github.com/moniquemoura/json-server
npm install
node server.js
```
